Control Help

This application was created as a framework. Each search utilizes eventtypes and tagging in order to more easily customize the integration of other data sources. This help section outlines the definition of the searches, and supplies a listing of the fields which may be used for reporting. Please take a look at the About page for more information on customization.

This control utilizes data from the `device_access` search macro.

USB Connection Events

The `device_access` macro is defined as:

tag=peripheral tag=communicate tag=connect $filter$ `device_access_filter`

In the event that there are filter-based modifications to the search that you'd like to employ permanently, you may edit the `device_access_filter` to affect the results. Further adjustments may require modifying the `device_access` macro itself.

Recommended Fields:

  • dest = destination host
