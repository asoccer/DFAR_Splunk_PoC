Control Help

This application was created as a framework. Each search utilizes eventtypes and tagging in order to more easily customize the integration of other data sources. This help section outlines the definition of the searches, and supplies a listing of the fields which may be used for reporting. Please take a look at the About page for more information on customization.

This control utilizes data from the `wireless_event` search macro.

Wireless Access Events

The `wireless_event` macro is defined as:

tag=wireless tag=ids $filter$ `wireless_event_filter`
   | eval dest=coalesce(dest, DST)
   | eval src=coalesce(SRC, src)
   | fillnull value=unknown src, dest, vendor, severity, signature

In the event that there are filter-based modifications to the search that you'd like to employ permanently, you may edit the `wireless_event_filter` to affect the results. Further adjustments may require modifying the `wireless_event` macro itself.

Recommended Fields:

  • src = source host
  • dest = destination host
  • vendor = the product vendor (e.g. AirDefense)
  • severity = the event severity
  • signature = a simple description of the event
